# codex-django Documentation

`codex-django` is the runtime package in the Codex Django stack.
This documentation focuses on installable runtime modules, with companion notes for projects generated by `codex-django-cli`:

- English documentation in [`docs/en/`](./en/README.md)
- Russian documentation in [`docs/ru/`](./ru/README.md)
- English API reference in [`docs/en/api/`](./en/api/README.md)

## Main Areas

- Guide layer: getting started, installation modes, runtime vs CLI, project structure, blueprint workflow
- Architecture layer: runtime modules and generated project boundaries
- API reference: English-only pages generated from source docstrings
