# codex-django Documentation

`codex-django` combines reusable Django runtime modules and a project scaffolding CLI for Codex projects.
This documentation is split into two parallel guides:

- English documentation in [`docs/en/`](./en/README.md)
- Russian documentation in [`docs/ru/`](./ru/README.md)

## Main Areas

- Runtime architecture: `core`, `system`, `notifications`, `booking`, `cabinet`, `showcase`
- CLI architecture: entrypoints, blueprints, engine, commands, generated project output
- API reference: generated separately from docstrings
